## Want to run it locally? 🔧
1. Clone the repo:
```bash
git clone https://github.com/VictorHumberto01/VictorPortfolio.git
```

2. Install dependencies:
```bash
npm install
```

3. Fire it up:
```bash
npm run dev
```

4. Open `http://localhost:3000` in your browser
